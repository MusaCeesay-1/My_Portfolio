## Run Locally

1. Clone the repository:

```bash
git clone https://github.com/MusaCeesay-1/My_Portfolio.git
cd My_Portfolio
```

2. Open `index.html` in your browser.

Optional (recommended): use a local server for cleaner asset loading.

Example with Python:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
