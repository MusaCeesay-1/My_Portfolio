# Roadmap

## Objective

Prioritize the next highest-impact portfolio moves.

## Workflow

1. Clarify target roles and desired outcomes.
2. Audit current portfolio strengths and gaps.
3. Propose prioritized initiatives with expected impact.
4. Define concrete next actions and sequencing.
5. Include decision rationale and tradeoffs.

## Output Rules

- Rank recommendations explicitly.
- Tie every recommendation to hiring outcomes.
- Balance quick wins with strategic depth.
- Flag dependencies and effort level.
