# SEO

## Objective

Improve discoverability and share-preview quality.

## Workflow

1. Review current metadata and page intent.
2. Draft concise, unique title and description content.
3. Ensure social tags and canonical references are correct.
4. Validate sitemap and robots coverage for relevant pages.
5. Document changes and expected SEO impact.

## Output Rules

- Avoid keyword stuffing.
- Keep metadata factual and specific.
- Maintain consistency with on-page content.
- Prefer deterministic checks over assumptions.
